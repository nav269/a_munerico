Ejercicio 1.Crecimiento poblacional con capacidad limitada

El crecimiento poblacional de especies con capacidad limitada puede modelarse mediante la ecuación:

$$ \frac{dN}{dt} = kN(N_M - N) $$

donde N es el tamaño de la población, NM es el número límite para la población, y k es una constante. Considera el caso donde NM = 5000, k = 0.000095 1/año, y N(0) = 100. Determina N para 0 ≤ t ≤ 20. Realiza una gráfica de N como función de t.

Ejercicio 2. Modelado del crecimiento de un tumor

El crecimiento de un tumor puede modelarse con la ecuación:

$$ \frac{dA}{dt} = \alpha A \left[1 - \left(\frac{A}{k}\right)^\nu\right] $$

donde A(t) es el área del tumor y $\alpha$, k, y $\nu$ son constantes. Resuelve la ecuación para 0 ≤ t ≤ 30 días, dado $\alpha$ = 0.8, k = 60, $\nu$ = 0.25, y A(0) = 1 mm2. Realiza una gráfica de A como función del tiempo.

Ejercicio 3. Velocidad de un objeto en caída libre

La velocidad de un objeto que cae libremente debido a la gravedad terrestre puede modelarse mediante la ecuación:

$$ m\frac{dv}{dt} = -mg + kv^2 $$

donde m es la masa del objeto, g = 9.81 m/s2, y k es una constante. Resuelve la ecuación para v para el caso m = 5 kg, k = 0.05 kg/m, 0 ≤ t ≤ 15 s y v(0) = 0 m/s. Realiza una gráfica de v como función del tiempo.
